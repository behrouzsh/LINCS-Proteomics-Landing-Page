LINCS & Related Proteomics Tools

See some of tools developed within the LINCS consortium and partners.

Tools to help you be more productive with LINCS and Non-LINCS proteomics datasets. For complete list of LINCS Workflows and Tools click here

PIR (The Protein Information Resource) is an integrated public bioinformatics resource to support genomic, proteomic and systems biology research and scientific studies.
piNet can be used to annotate, map and analyze a set of peptide moieties (including post-translationally modified peptides) identified by either targeted (SRM type) or SWATH (discovery) proteomic assays...
Panorama Repository Software for Targeted Proteomics Assays from Skyline.
iLINCS (Integrative LINCS) is an integrative web platform for analysis of LINCS data and signatures. The portal provides biologists-friendly user interfaces for analyzing transcriptomics and proteomics LINCS datasets...
Enrichr is sn intuitive web-based gene list enrichment analysis tool for inferring knowledge about an input gene set by comparing it to annotated gene sets representing prior biological knowledge with enrichment analysis.
piLINCS has been developed as part of the BD2K-LINCS Data Coordination and Integration Center to provide interactive and API access to proteomics data generated by the PCCSE center.
PhosphoSitePlus Freely accessible on-line systems biology resource devoted to commonly studied protein post-translational modifications...
HMS LINCS Database The Database contains all publicly available HMS LINCS datasets and information for each dataset about experimental reagents and experimental and data analysis protocols....
LINCS DATA PORTAL Provides unified interface for searching all LINCS dataset packages and entities.
PRO (PRotein Ontology) provides an ontological representation of protein-related entities by explicitly defining them and showing the relationships between them.
iPTMnet is a bioinformatics resource for integrated understanding of protein post-translational modifications (PTMs) in systems biology context.
iProLINK (integrated Protein Literature, INformation and Knowledge) is a resource with access to text mining tools and annotated corpora developed in house.

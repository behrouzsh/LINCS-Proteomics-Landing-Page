################################################################
Dataset Information
################################################################

Dataset Name: 
Proteomics Dataset (March 2016)
Dataset Description: 
The overall goal is to develop robust cellular signatures for drug-induced toxicity and toxicity mitigation. These signatures will be established by exposing cells to drugs and drug combinations in culture followed by Mass Spectrometry.

Dataset Contents:
1) Level-3-R2016-03-08.txt
Location: In package
Description: Mass Spectrometry data files

2) Small_Molecule_Metadata.txt
Location: In package
Description: Metadata file 1

3) Primary_Cell_Metadata.txt
Location: In package
Description: Metadata file 2

4) Antibody_Metadata.txt
Location: In package
Description: Metadata file 3

Metadata Information
################################################################

Metadata information regarding the entities used in the experiments is included in the accompanied metadata. A metadata file per entity category is included in the package. The metadata for all the cell lines that were used in the dataset are included in the Cell_Lines_Metadata.txt file.

Descriptions for each metadata field can be found here: http://www.lincsproject.org/data/data-standards/
